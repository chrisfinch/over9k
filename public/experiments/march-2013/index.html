This page was a built as an effort to understand and implement d3.js' .d3.geo funtionality and line simplification for SVG based plotting.

I initialy found Mike Bostocks example using the Visvalingam Algorithm for line simplicfication at http://bost.ocks.org/mike/simplify/ which led me, after some research, to Jason Davies extention of the implementation at http://www.jasondavies.com/maps/simplify/ and finally to the d3.simplify plugin: https://github.com/d3/d3-plugins

The map implementation is built by me but all work on the simplification algorithm itself is not my own.

For further work in this area I will investigate simplification with TopoJSON to try and create a zooming map.
